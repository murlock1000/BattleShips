If you want to play the game
Launch Laivu_m_FE/Release/Laivu_m_FE.exe

If you want to edit the code, you will
need library SFML which can be found at 
https://www.sfml-dev.org/download/sfml/2.5.1/

Installation tutorial:
For C++ https://www.sfml-dev.org/tutorials/2.5/start-cb.php
For VS https://www.sfml-dev.org/tutorials/2.5/start-vc.php
For Linux(?) https://www.sfml-dev.org/tutorials/2.5/start-linux.php
